Plot Based Movie Reccomendations
Using LLMS ANd Vector Databases

In this project, I set up a Retrieval-Augmented Generation (RAG) system powered by a Chroma vector database and a Large Language Model (LLM). Text embeddings were created from the plot descriptions of the IMDb dataset using sentence transformers from Hugging Face. These embeddings were then loaded into a Chroma vector store. Investigating RAG And LLMS Potential in Analyzing
Sport Performance Leveraging Football Match Commentary

In this project, I explored how Retrieval-Augmented Generation (RAG) and Large Language Models (LLMs) can be applied to sports analytics using football match commentary. Audio commentary was first transcribed with OpenAI’s Whisper model to generate structured text data. These transcripts were embedded and stored in a vector database, forming the basis of a RAG system. By querying this system with an LLM such as LLaMA, I was able to generate player-specific performance summaries, offering a novel approach to extracting insights from unstructured broadcast commentary.

Tech stack: Whisper, Chroma, LangChain, LLaMA, Hugging Face

Creating a Song Similarity Index:Setting up a Vector Database From The 5 Miliion Song Lyrics Dataset Using Chroma, Langchain And GTE-large Model

In this project, I developed a lyrics-based song recommendation system that measures similarity by analyzing lyrical content and emotional context, rather than relying solely on metadata such as BPM or loudness. Using the 5 Million Song Lyrics dataset (over 5.9M rows of titles, artists, tags, years, and lyrics), I loaded and chunked the data with LangChain’s DataFrameLoader and RecursiveCharacterTextSplitter. Text chunks were then embedded with the thenlper/gte-large model via the Sentence Transformers library and stored in a Chroma vector database. With similarity search and max marginal relevance search, the system generates song recommendations based on lyrical similarity. This approach demonstrates how Natural Language Processing and vector search can enhance music discovery, and lays the foundation for future work such as LLM-driven playlist generation and scaling to larger datasets.

Tech stack: LangChain, Chroma, Sentence Transformers (thenlper/gte-large), Hugging Face

Creating a Movie Similarity Index:Setting up a Vector Database From The IMDB Dataset Using Chroma, Langchain And Sentence Transformers

In this project, I built a movie recommendation system that identifies similar films based on plot descriptions and related metadata. Using the IMDb Movies dataset (10k+ entries including title, release date, genre, cast, and overview), I created a combined text field merging plot, crew, and genre information. LangChain handled preprocessing and chunking, while the all-mpnet-base-v2 model from Hugging Face was used to generate embeddings via Sentence Transformers. These embeddings were stored in a Chroma vector database and later migrated to a Pinecone cloud vector store for scalability. Similarity search and max marginal relevance search enabled recommendations based on narrative context rather than surface-level attributes. This approach highlights the potential of transformers and vector databases in enhancing content discovery, with future directions including larger-scale databases and LLM-driven binge-list curation.

Tech stack: LangChain, Chroma, Pinecone, Sentence Transformers (all-mpnet-base-v2), Hugging Face

Automated Sport Transcription Using FFMPEG and OpenAI Whisper Models

In this project, I developed a system to transcribe sports videos and audio into structured text commentary. Video and audio sources were processed using the yt_dlp library for downloading content and FFmpeg for audio extraction. OpenAI’s Whisper model was then used to transcribe the extracted audio into text, enabling future analysis with LLMs to derive player performance insights. Two inference functions were created: one to transcribe from video or audio file paths and another to transcribe directly from YouTube links, both successfully tested on sample football matches.

Tech stack: yt_dlp, FFmpeg, OpenAI Whisper
